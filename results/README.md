# Results

This directory contains the outputs generated from the SARS-CoV-2 variant analysis pipeline.

## Files

**annotated_variants.vcf**  
VCF file containing variants annotated with predicted functional effects.

**spike_mutations.vcf**  
Subset of variants located in the spike gene region.

**snpEff_genes.txt**  
Gene annotation output generated using SnpEff.

**variant_count.txt**  
Summary of the total number of variants detected.

**mean_coverage.txt**  
Average sequencing coverage across the genome.

**mapping_stats.txt**  
Statistics describing the alignment of reads to the reference genome.

## figures

This subdirectory contains plots generated during the analysis.

Figures include:

- Genome coverage plot
- Manhattan plot of variant positions
- Allele frequency distribution

These visualizations help interpret sequencing depth and genomic variation across the viral genome.
