# Results

This directory contains the outputs generated by the variant analysis pipeline.

## variants.vcf

Variant Call Format (VCF) file containing the detected variants relative to the reference genome.

## coverage.txt

Coverage statistics across the genome.

## figures

This subdirectory contains plots generated during the analysis.

Figures include:

- Genome coverage plot
- Manhattan plot of variant positions
- Allele frequency distribution

These visualizations help interpret sequencing depth and genomic variation across the viral genome.
