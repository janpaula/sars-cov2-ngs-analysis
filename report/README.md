# Analysis Report

This directory contains the final report of the SARS-CoV-2 genomic analysis.

## analysis.pdf

The report summarizes the complete bioinformatics workflow, including:

- Description of the dataset (2020 SARS-CoV-2 sequencing data)
- Bioinformatics pipeline
- Variant detection results
- Coverage analysis
- Graphical visualizations

The report was generated from an RMarkdown document and exported as a PDF.
