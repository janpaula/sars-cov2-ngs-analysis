# Scripts

This directory contains the scripts used during the SARS-CoV-2 variant analysis.

## variant_analysis.R

R script used to perform variant analysis and generate the figures and tables included in the final report.

The script processes variant data and produces:

- variant summaries
- coverage statistics
- graphical visualizations
- tables used in the report

## sra_explorer_fastq_download.sh

Shell script generated using SRA Explorer to download sequencing data (FASTQ files) from the Sequence Read Archive (SRA). of a bioinformatics analysis of SARS-CoV-2 sequencing data from 2020, demonstrating a standard pipeline for viral genome variant detection.
